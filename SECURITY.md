# Security Policy

## Supported Versions

This is the list of versions of vue-slick-ts which are
currently being supported with security updates. <br />

Version &lt;= 0.x.x doesn't have support types, events, methods

| Version            | Supported          |
| ------------------ | ------------------ |
| 1.x.x              | :white_check_mark: |
| &lt;= 0.x.x        | :x:                |
